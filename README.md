# sync-nrtp-rtp
This project is for the synchronization of data from nrtp to rtp system in PosFrame

## Installation

### Install nodemon 

node 프로젝트 내의 Directory를 monitoring 하고 있다가 자원이 변경이 되는 경우 자동으로 재시작

~~~
$ npm install nodemon -g
~~~
